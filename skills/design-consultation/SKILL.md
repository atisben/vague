## Preamble

```bash
export BS_SKILL_NAME="design-consultation"
source "$(dirname "$0")/../_preamble.sh"
```

---

## Step 1: Understand the Product

Ask one question at a time:

1. "Describe your product in one sentence — what it does and who it's for."
2. "What three adjectives should someone feel when using it? (e.g., calm/powerful/playful)"
3. "Name 2-3 products whose design you admire. What specifically do you like about them?"
4. "What should the design absolutely NOT feel like? Any anti-patterns?"
5. "Is this primarily desktop, mobile, or both?"

---

## Step 2: Landscape Research

Run 2-3 web searches to understand the visual conventions and expectations in this product space. Look for:
- Dominant design patterns in this category
- Color palettes and typography trends
- What the best-in-class products look like
- What feels dated or overdone

Synthesize findings: "In this space, [conventions]. An opportunity to stand out: [differentiation angle]."

---

## Step 3: Design System Proposal

Present a complete design system proposal via AskUserQuestion:

```
PROPOSED DESIGN SYSTEM

Aesthetic direction: [3-5 sentences describing the visual personality]

Typography:
  Display:  [font name] — [why it fits]
  Body:     [font name] — [why it fits]
  Mono:     [font name, if needed]
  Scale:    [xs/sm/base/lg/xl/2xl sizes in px or rem]

Color:
  Primary:    [hex] — [name and use]
  Secondary:  [hex] — [name and use]
  Accent:     [hex] — [name and use]
  Background: [hex]
  Surface:    [hex]
  Text:       [hex]
  Muted:      [hex]
  Success:    [hex]
  Warning:    [hex]
  Error:      [hex]

Spacing: [base unit and scale — e.g., 4px base, 4/8/12/16/24/32/48/64]

Border radius: [values for sm/md/lg/full]

Shadows: [sm/md/lg definitions]

Motion:
  Duration:  [fast/normal/slow in ms]
  Easing:    [standard curve]

→ A) Looks good, proceed   B) Adjust [what?]   C) Show me alternatives
```

---

## Step 4: Alternatives (if requested)

If user asks for alternatives, generate 2 more distinct directions:
- Direction B: [contrasting aesthetic]
- Direction C: [different mood entirely]

Present each with the same structure as Step 3.

---

## Step 5: Font + Color Preview Pages

Generate HTML preview files:

**fonts.html** — shows every typeface at every scale, with sample headings and body text
**colors.html** — shows every color swatch with hex, name, usage, and contrast pairs

```bash
eval "$(~/.bastack/bin/bs-slug 2>/dev/null)"
mkdir -p ~/.bastack/projects/$SLUG/design-previews
```

Write the HTML files to `~/.bastack/projects/$SLUG/design-previews/`. Then open them:
```bash
open ~/.bastack/projects/$SLUG/design-previews/fonts.html
open ~/.bastack/projects/$SLUG/design-previews/colors.html
```

---

## Step 6: Write DESIGN.md

Save to the project root as `DESIGN.md`:

```markdown
# Design System

> Auto-generated by bastack /design-consultation on [date]

## Aesthetic Direction
[3-5 sentence description of the visual personality]

## Typography
| Role    | Font     | Fallback Stack | Notes |
|---------|----------|----------------|-------|
| Display | ...      | ...            | ...   |
| Body    | ...      | ...            | ...   |

### Type Scale
| Token | Size  | Line Height | Weight |
|-------|-------|-------------|--------|
| xs    | 12px  | 16px        | 400    |
| sm    | 14px  | 20px        | 400    |
| base  | 16px  | 24px        | 400    |
...

## Colors
| Token      | Hex     | Usage               |
|------------|---------|---------------------|
| primary    | #...    | CTAs, links         |
...

## Spacing
Base unit: 4px
Scale: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128

## Border Radius
| Token | Value  |
|-------|--------|
| sm    | 4px    |
| md    | 8px    |
| lg    | 16px   |
| full  | 9999px |

## Shadows
[definitions]

## Motion
| Token  | Duration | Easing        |
|--------|----------|---------------|
| fast   | 100ms    | ease-out      |
| normal | 200ms    | ease-in-out   |
| slow   | 400ms    | ease-in-out   |

## Anti-patterns
[What this design should never look like]
```

Commit:
```bash
git add DESIGN.md && git commit -m "design: add design system via /design-consultation"
```
